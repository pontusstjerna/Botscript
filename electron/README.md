# Electron boilerplate (at least for now)
Please download and install NPM first! 

## How to run
Run these two commands __simultaneously__ in different console tabs.

```bash
$ npm run hot-server
$ npm run start-hot
```

or run two servers with one command

```bash
$ npm run dev
```
